#### 1.创建用户

```linux
useradd 用户名称
cd /home ~~可以查看创建用户的信息
```

#### 2.修改密码

```
passwd 用户名称
```



#### 3.删除用户

```
userdel -r 用户名
```

####  4.查看用户ID信息

```
id userone
uid=501(userone) gid=501(userone) groups=501(userone)
用户ID            用户组ID          用户组名称
```

#### 5.切换用户

```
su userone
高权限切低权限用户不用输入密码
```

#### 6.新建组

```
groupadd zh
```

#### 7.新建用户指定分组

```
useradd -g zh usertwo
```

#### 修改用户组信息

```
useromd -g zh userone
```
